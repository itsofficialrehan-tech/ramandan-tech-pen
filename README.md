🌙 Ramadan Simulator — Hand Gesture Drawing Experience

Ramadan Simulator is a browser-based interactive experience built with HTML5 Canvas and MediaPipe Hands.
It uses real-time hand-tracking to let users draw glowing dark-golden strokes in the air using their index finger, creating a spiritual and immersive Ramadan-themed visual.

✨ Features

🖐 Real-time hand tracking using MediaPipe Hands

✍ Air-drawing system controlled by index finger

🎮 Control key = Draw mode

🖤 Black cinematic mask with body cut-out effect

🌟 Dark golden glowing pen with smooth motion

🎥 Live camera feed mirrored for natural interaction

💻 Runs fully in browser (No backend required)

🎮 Controls

CTRL (Hold) → Start drawing

Release CTRL → Stop drawing

SPACE → Clear all drawings

🛠 Tech Stack

HTML5

CSS

JavaScript

MediaPipe Hands

Canvas API

📸 Use Case

Perfect for:

Ramadan themed creative projects

Gesture-based UI experiments

Interactive art installations

Learning MediaPipe & Canvas together

🚀 How to Run

Download or clone the repository

Open index.html in a modern browser (Chrome recommended)

Allow camera access

Use your index finger + CTRL key to draw in the air
